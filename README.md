## Music UI Website

A modern and responsive music UI website designed to deliver a clean layout, smooth interactions, and an engaging user experience inspired by popular music streaming platforms.

## Features

Modern dark-themed user interface

Sidebar navigation with playlists and menu options

Now Playing section with album art and track details

Play and pause music controls

Interactive progress bar with seek functionality

Keyboard support for play/pause

Smooth hover effects and transitions

Responsive design for different screen sizes

## Technologies Used

HTML5

CSS3

JavaScript

## Project Purpose

This project was built to demonstrate front-end development skills, focusing on UI design, layout structuring, animations, and basic audio player functionality using vanilla JavaScript.
